# Common infrastructure

### DOM Trees
只有给内容属性赋予了一个新值，才能称之为改变，赋予一个同样的值不叫改变它。

### Plugins

插件只是使用 UA 的渲染文档对象，它们并不作为文档的子浏览器环境也不会引入任何新的节点对象到 Document 的 `DOM` 中。

### Elements
所有的带 `name` 属性的元素：
`a` `button` `embed` `form` `frame` `frameset` `img` `input` `map` `meta` `object` `select` `textarea` 。
